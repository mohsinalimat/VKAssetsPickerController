# VKAssetsPickerController
An Image Picker For iOS.

一个图片选择器，简单封装，简单易用。具体参考示例代码的使用方式。

功能：

* 支持限定张数的图片选中，超过设定的张数时，回调代理方法处理。

* 支持记录上次选中的照片，下次再次选择时，依然保持选中状态。

* 支持图片选择时的预览大图功能。
